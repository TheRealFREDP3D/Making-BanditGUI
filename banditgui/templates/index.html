Bandit Game Assistant

Welcome to BanditGUI!

This application helps you learn about and interact with the OverTheWire Bandit CTF challenges.

You can ask for help or use these commands in the chat:

  • help - Get general information about Bandit wargame
  • info - Check connection status and current level
  • level - Get instructions for the current level
  • quit - Disconnect and exit the game

Use the terminal on the right to execute commands on the Bandit server (bandit.labs.overthewire.org).

This is a real SSH connection - all commands are sent directly to the server.

Type help, info, or level in the chat for assistance.
